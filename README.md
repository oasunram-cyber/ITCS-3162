## Getting Started

### Requirements

- Python 3.9 or later
- Jupyter Notebook or JupyterLab
- Common data mining libraries:
  - `numpy`
  - `pandas`
  - `scikit-learn`
  - `matplotlib`

Install the dependencies with:

```bash
pip install numpy pandas scikit-learn matplotlib jupyter
```

### Running the Notebooks

**Recommended — Google Colab:** Click any **Open in Colab** badge in the table
above to launch that notebook directly in the browser. No local setup required.
